# 선택 이유

- 기본 도메인으로 바로 시작할 수 있어요.
- 정적 export와 조합이 단순해요.
- 비용 없이 개인 아카이브를 운영하기 좋아요.

## 남은 일

실제 공개 배포 전에는 Actions 워크플로와 Pages 설정만 추가하면 돼요.
